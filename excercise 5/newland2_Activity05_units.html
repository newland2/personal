When the window height changes the 5vh text changes as well. When the window is tall the text is big and vice versa. The other text stays a constant size.

Eli Scott Newland(Horizontal)

When the window width changes the 5vh text changes as well. When the window is wider the text is bigger and vice versa. The other text stays a constant size.
